Certification Overview

The AWS Certified Cloud Practitioner (CLF-C02) credential establishes your basic understanding of Amazon Web Services (AWS). It is a vital milestone for beginners and non-technical business professionals looking to gain strong technical credibility.

As cloud infrastructure budgets expand across global organizations, there is a severe shortage of certified personnel who can navigate basic deployment principles, manage billing systems, comprehend Shared Responsibility security models, and catalog core compute, database, and storage platforms.

Syllabus Outline & Modules

Explore the comprehensive domains covered in the updated CLF-C02 syllabus. Each module includes video classes, active labs, and mock exams.

  • Defining the AWS Cloud value proposition and cost benefits
  • Understanding cloud delivery models: Infrastructure as a Service (IaaS), Platform as a Service (PaaS), Software as a Service (SaaS)
  • Reviewing the AWS Global Infrastructure: Availability Zones (AZs), AWS Regions, and Edge Locations

  • AWS Shared Responsibility Security Model: Customer controls vs. AWS controls
  • AWS Identity and Access Management (IAM): Users, groups, roles, permissions, and Multi-Factor Authentication (MFA)
  • Encryption strategies, AWS Key Management Service (KMS), and compliance tools (AWS Artifact)

  • Compute Services: Amazon Elastic Compute Cloud (EC2), AWS Lambda, and AWS Elastic Beanstalk
  • Storage Platforms: Amazon Simple Storage Service (S3) and Elastic Block Store (EBS)
  • Databases: Amazon Relational Database Service (RDS), Amazon DynamoDB, and Amazon Aurora
  • Networking: Amazon Virtual Private Cloud (VPC), Route 53, and CloudFront Content Delivery Network (CDN)

  • Comparing AWS pricing models (On-Demand, Savings Plans, Spot Instances, Reserved Instances)
  • Budgeting tools: AWS Budgets, Cost Explorer, and the AWS Pricing Calculator
  • AWS Support Plans: Developer, Business, and Enterprise tiers and characteristics
